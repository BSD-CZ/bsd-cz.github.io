# About BSD-CZ

BSD-CZ is a Czech FreeBSD community dedicated to practical BSD work — drivers, embedded systems, and hardware enablement.

## Our mission

FreeBSD is a rock-solid operating system, but hardware support sometimes lags behind Linux. We aim to close that gap by:

1. Writing drivers for unsupported hardware
2. Testing and documenting FreeBSD on modern machines
3. Building embedded FreeBSD images for ARM and RISC-V platforms
4. Contributing patches upstream to the FreeBSD project

## Why FreeBSD?

- **Clean codebase** — consistent style, well-documented kernel interfaces
- **Licensing** — permissive BSD license, friendly to commercial and embedded use
- **Networking** — FreeBSD's network stack remains one of the best in the industry
- **ZFS** — first-class ZFS support out of the box
- **Jails & bhyve** — lightweight virtualization without the bloat
